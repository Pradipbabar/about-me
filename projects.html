Projects

    • Automated Deployment with Jenkins and Tomcat

      In this project, I worked with a team to implement an automated deployment pipeline using Jenkins and Tomcat. The pipeline included the following stages:

      • Code checkout from Git repository
      • Build and test using Maven
      • Deployment to a test environment
      • Automated functional and performance testing
      • Deployment to a production environment upon successful testing

      We also utilized the Jenkins Pipeline plugin to define the pipeline as code and make it more maintainable. Additionally, we implemented a rolling deployment strategy to minimize downtime during production deployments.

      The implementation of this pipeline resulted in faster and more reliable deployments, and a significant reduction in manual errors. It also allowed us to easily roll back to previous versions if any issue occurs.

    • Multi-Service Application with Docker Compose

      In this project, I developed a multi-service web application using Docker Compose. The application consisted of several microservices, including a web frontend, a RESTful API, and a database service. Each service was containerized using Docker and managed using Compose.

      I utilized Docker Compose to define and run the application’s services in a single command, making it easy to set up and manage the application in different environments. I also created a single container for the application and used a .yml file to define the services, networks and volumes.

      The use of Docker Compose allowed for easy scaling and management of the application’s services, and simplified the process of deploying the application to different environments, including development, staging, and production. Additionally, I also used a private registry to store the images to keep them secure and only accessible for authorized users.

üõ†Ô∏è Commands

  • configure
    Set your Gemini API access key in environment. 
    gemini-cli configure -k YOUR_ACCESS_KEY
  • chat
    Send a prompt and receive a chat response. 
    gemini-cli chat -p "Your prompt text" -s output.txt
    -s flag saves the output to a file (optional).
  • explain
    Generate explanations from input file. 
    gemini-cli explain -f input.txt -s output.txt

üöÄ Quick Usage

  1. Run gemini-cli configure to store your key.
  2. Use gemini-cli chat to have interactive conversations.
  3. Use gemini-cli explain to process text files.

The access key is saved in GEMINI_AI_ACCESS_KEY for subsequent commands.

üöÄ Quick Start

Create a root main.tf and reference the module:

module "resource_master" {
  source      = "Pradipbabar/resource-master/aws/"
  enable_vpc  = true
  enable_ec2  = true
  enable_rds  = true
  enable_s3   = true
  # enable_cloudwatch = true
  # enable_dynamodb  = true
}

Then run:

terraform init
terraform apply

To tear down:

terraform destroy
